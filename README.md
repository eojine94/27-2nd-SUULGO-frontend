## 적용 기술 및 구현 기능

### 적용 기술

- Front-End: React.js, styled-componets, React Router
- Back-End: Python, Django web framework, MySQL
- Common: AWS

### 소통 툴

- Git, Github, Slack, Notion

### 구현기능

- 네브바
  - 카카오 소셜 로그인
- 메인페이지
- 설문조사 페이지
- 회원 리스트 페이지
- 회원 상세 페이지
  - 약속잡기
  - 일치 태그 표시
- 마이 페이지
